# V18.3.1 — Official Links and Reference Sources

This release separates entity identity from research provenance.

- `officialLinks[]` contains only resources owned by or directly representing the entity.
- `referenceSources[]` records podcasts, hearings, government reports, books, documents, and articles used to create or expand the entity.
- Organizations without a verified official presence display a neutral explanatory note.
- Existing source metadata was migrated without discarding URLs.
- Import templates and validation now enforce the distinction before the Research Accelerator is introduced.
