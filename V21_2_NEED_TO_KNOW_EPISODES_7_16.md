# V21.2 — Need to Know Podcast Integration (Batch 3)

Content-only production ingestion of Need to Know Episodes #7–16 using the validated V21 workflow. The release adds ten complete episode records, forty source-grounded principal claims, ten publication timeline events, and focused relationships to existing canonical people, organizations, hearings, documents, cases, publications, and topics.

Episode #13 is the distinct short-form “Bryce’s Rant” release. Episode #14 is the Luis Elizondo discussion associated with the May 17, 2022 House UAP hearing. This preserves the official release sequence without duplicating an episode entity.

Every episode uses its direct official YouTube video as the primary media reference. The official Need to Know YouTube collection remains the secondary reference.

No architecture, schema, rendering, template, navigation, routing, UI, styling, or knowledge-graph engine changes were made.
