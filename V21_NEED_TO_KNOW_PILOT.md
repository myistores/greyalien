# V21 — Need to Know Podcast Integration Pilot

Content-only production ingestion of Need to Know Episode #1, “2021 Top Ten.” The episode reuses the existing canonical Need to Know series, Ross Coulthart, Bryce Zabel, and topic entities. Relationships are limited to the official episode description and established host roles.
